The 1st Workshop on

Detecting Objects in Aerial Images

in conjunction with IEEE CVPR 2019 June 16, 2019, Long Beach, California.

  • 8:40 - 8:50 Welcome, Introduction

  • 9:30 - 9:50 Oral Presentation: Multi-Scale Weighted Branch Network for Remote Sensing Image Classification
  • 9:50 - 10:10 Coffee Break
  • 10:10 - 10:30 Oral Presentation: Window Detection in Facades for Aerial Texture Files of 3D CityGML Models
  • 10:30 - 10:50 Oral Presentation: iSAID: A Large-scale Dataset for Instance Segmentation in Aerial Images
  • 10:50 - 11:10 Oral Presentation: ArcticNet: A Deep Learning Solution to Classify the Arctic Area
  • 11:10 - 11:30 Oral Presentation: The Aerial Elephant Dataset: A New Public Benchmark for Aerial Object Detection
  • 11:30 - 11:50 Oral Presentation: Multi-Cue Vehicle Detection for Semantic Video Compression in Georegistered Aerial Videos
  • 11:50 - 12:10 Award ceremony for the challenge winners
